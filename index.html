LDS Toolbox

Tools for members of the LDS Church

View on GitHub Download .zip Download .tar.gz

LDS Toolbox is an open-source, real-time suite of tools for members of the LDS Church built with Meteor and based on Meteoric Ionic as a template. It also makes use of the lds.org API Documentation for some of the data.

These tools augment tools already available through LDS Tools, Leader and Clerk Resourced (LCR), and lds.org.

If you'd like to be part of the project, please see if there's anything you can help with and let us know. Also, feel free to view/submit issues and milestones.
